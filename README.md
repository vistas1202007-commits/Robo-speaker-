# 🤖 Robo Speaker 2.0

A simple Python application that converts your text into spoken words using system's built-in voice engine. Just type and listen!

---

## 📌 Features

- 🔊 Converts typed text to speech instantly
- 🔁 Continuous loop — speak multiple sentences
- 👋 Type `q` to exit with a goodbye message
- ⚡ Lightweight — no heavy libraries needed

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Python 3.x | Core language |
| os module | System speech command |

---

## 📦 Installation

**Step 1: Clone the repository**
```bash
git clone https://github.com/vistas1202007-commits/robo-speaker.git
cd robo-speaker
```

**Step 2: Run the application**
```bash
python robo.py
```

> No extra libraries needed! Uses Python's built-in `os` module.

---

## 🚀 How to Use

1. Run the script
2. Type any text you want to hear
3. Press **Enter** to speak
4. Type **`q`** to exit
